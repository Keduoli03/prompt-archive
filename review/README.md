# 评审流程说明

## 工作流

```
定时任务抓取 → review/ 评审区
        ↓
   你人工审核
        ↓
  ✅ 通过 → 移入 prompts/ 对应分类
  ❌ 淘汰 → 删除或移入 rejected/
```

## 评审标准

- [ ] 提示词结构完整（主体+风格+构图+参数）
- [ ] 有明确风格描述，不是泛泛而谈
- [ ] 适合 AI 图片生成，不是普通描述文字
- [ ] 有独特亮点（构图手法/材质质感/创意角度）

## 操作方法

**通过** - 移到正式目录：
```bash
mv review/YYYYMMDD-xxxx.md prompts/对应分类/
```

**淘汰** - 移到回收站：
```bash
mv review/YYYYMMDD-xxxx.md rejected/
```
