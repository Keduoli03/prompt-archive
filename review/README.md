# 评审流程说明

## 重要规则

- ❌ 禁止自己编写提示词
- ✅ 只收录**他人真实发布**的提示词
- ✅ 必须有原文链接
- ✅ 必须有作者信息
- ✅ 最好有效果图（附带说明）
- ✅ 收录后更新 `index/INDEX.md`

## 收录标准

- [ ] 有原始发布链接（Twitter/X、小红书、B站等）
- [ ] 有作者信息（用户名/作者名）
- [ ] 提示词结构完整，不是片段
- [ ] 有实际效果验证（截图/描述）
- [ ] 风格独特，非泛泛描述

## 操作方法

**通过** → 移到 `prompts/对应分类/`，附上效果图到 `assets/`

**淘汰** → 删除
