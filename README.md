# Prompt Archive

精心筛选的 AI 图片生成提示词收藏库，按主题分类整理。

## 收录范围

- AI 图片生成提示词（Midjourney、Stable Diffusion 等）
- 必须为真实发布过的内容，拒绝编造
- 必须包含原始发布链接、作者信息、完整或接近完整的原文

## 目录结构

```
prompt-archive/
├── prompts/              # 正式收录提示词
│   ├── portrait/        # 人物肖像
│   ├── landscape/       # 风景场景
│   ├── concept-art/     # 概念艺术
│   ├── character-design/# 角色设计
│   ├── movie-poster/    # 电影海报
│   └── other/           # 其他
├── index/
│   └── INDEX.md         # 分类索引
└── README.md
```

## 标签体系

### 质量标签

| 标签 | 说明 |
|------|------|
| `质量-优质` | 独特风格、参数完整、辨识度高 |
| `质量-中等` | 可用但较常见，或部分参数缺失 |
| `质量-一般` | 摘要型、泛用型、参考性内容 |

### 状态标签

| 标签 | 说明 |
|------|------|
| `prompt-review` | 待审状态（新增时自动添加） |
| `收录` | 已通过审批并收录至分类 |
| `内容不符` | 内容与描述不符或来源无法核实 |
| `低质量` | 质量过低，不符合收录标准 |

## 贡献流程

本库通过 GitHub Issue 进行提示词筛选，流程如下：

1. **提交候选**：通过 Issue 提交待审提示词（可手动或自动抓取）
2. **质量评级**：由维护者手动打上质量标签
3. **审批关闭**：添加 `收录` 标签（如适用）并关闭 Issue
4. **自动入库**：系统自动将通过的提示词写入对应分类目录

> **注意**：仅仓库所有者（Keduoli03）关闭的 Issue 会被系统自动入库。

## 收录标准

1. 有完整或接近完整的提示词原文
2. 有具体帖子或网页链接（可验证来源）
3. 有明确的作者信息
4. 风格描述清晰
5. 效果图：有则更佳，非强制要求

## 文件命名

```
prompts/分类/名称-YYYYMMDD.md
```

名称在前，日期在后，精确到年月日。

## 分类说明

| 分类 | 说明 |
|------|------|
| portrait | 人物肖像、头像、半身像 |
| landscape | 风景、环境、场景 |
| concept-art | 概念艺术、游戏原画 |
| character-design | 角色设计、NPC、怪物 |
| movie-poster | 电影海报、书籍封面 |
| poster-promotional | 宣传海报 |
| other | 其他混合风格 |
